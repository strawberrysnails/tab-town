# Tab Town
* Inspired by my website name, snails.town
* A custom new tab page Firefox extension made for personal use.
* This was created and intended for my own personal use, if you want to fork and add features, feel free.

# Usage
Open a new tab. Scroll or use the tab buttons to switch pages. Click ⚙ to open settings and add/remove bookmarks.

# Installation
Submit the zipped extension to addons.mozilla.org as unlisted, wait for signing, then install the downloaded .xpi.
