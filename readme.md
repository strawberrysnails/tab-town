Zip up your extension folder contents (select all files inside the folder, not the folder itself)
Go to addons.mozilla.org and create a Mozilla account if you don't have one
Submit the zip — when asked about distribution, choose "On My Own" so it stays unlisted/private
Mozilla runs an automated review and signs it, usually within minutes
Download the signed .xpi file they give you back
Drag and drop it into Firefox, or go to about:addons → gear icon → "Install Add-on From File"
